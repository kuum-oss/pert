**AI EPUB Translator** — это мощное десктопное приложение для автоматического и ручного перевода электронных книг в формате `.epub` с использованием нейросетей Google Gemini (2.0 Flash, 1.5 Pro). Программа сохраняет оригинальную HTML-разметку, стили и ссылки, позволяя локализовать книги без потери их структуры.

---

## ✨ Ключевые возможности

* 🤖 **Умный перевод:** Интеграция с Google Gemini API. Перевод с учетом контекста, без поломки HTML-тегов (`<p>`, `<div>`, `<a>`).
* 🎨 **Динамические настройки ИИ:** Изменение стиля перевода (Художественный, Технический, Адаптированный) и настройка глоссария в реальном времени.
* 💾 **Автосохранение сессий:** Программа запоминает, на какой главе вы остановились. Вы можете закрыть приложение и продолжить перевод позже с того же места!
* 💰 **Счетчик токенов и стоимости:** Встроенный калькулятор потраченных токенов и примерной стоимости API прямо в интерфейсе.
* 🖼️ **Предпросмотр обложки:** Автоматическое извлечение и отображение обложки книги при загрузке.
* 💻 **Продвинутый редактор кода:** Встроенный темный редактор с подсветкой синтаксиса HTML для ручных правок.
* 📚 **Локальная библиотека:** Удобное хранение и экспорт готовых книг в один клик.

## 🛠 Технологический стек

* **Язык:** Java (Swing)
* **Нейросеть:** [Google Gemini API](https://ai.google.dev/)
* **Сеть:** OkHttp3
* **JSON Парсинг:** Gson
* **Редактор:** RSyntaxTextArea (с темной темой)

## 🚀 Как начать работу

### Требования

* Установленный [Java Development Kit (JDK) 11](https://adoptium.net/) или новее.
* Бесплатный ключ API от [Google AI Studio](https://aistudio.google.com/app/apikey).

### Установка и запуск

1. Склонируйте репозиторий:

```bash
git clone https://github.com/ВАШ-ЮЗЕРНЕЙМ/epub-translator.git
```

2. Откройте проект в вашей любимой IDE (IntelliJ IDEA, Eclipse).
3. Убедитесь, что подключены все зависимости (OkHttp, Gson, RSyntaxTextArea). Если используете Maven/Gradle, обновите конфигурацию.
4. Запустите класс `Main.java`.

### Использование

1. Запустите программу.
2. Введите свой **Gemini API Key** (кнопка `🔑 ВВЕСТИ КЛЮЧ AI`).
3. Перетащите файл `.epub` в левое окно программы (Drag & Drop).
4. Выберите стиль перевода и при необходимости заполните глоссарий (например, `Jon Snow=Джон Сноу`).
5. Нажмите **«🚀 АВТО»** для полностью автоматического перевода всей книги или **«✨ AI ПЕРЕВОД»** для пошаговой работы.
6. По завершении нажмите **«💾 СОХРАНИТЬ КНИГУ»**.

## ⚠️ Важные примечания

* **Лимиты API:** Если вы используете бесплатный тариф Gemini, будьте готовы к возможным задержкам (ошибка 429 - Too Many Requests). В программу встроена система автоматического ожидания и повторных запросов.
* **Сохранность тегов:** ИИ проинструктирован не трогать системные теги, но всегда рекомендуется бегло просматривать переведенный HTML-код в правом окне перед применением.
